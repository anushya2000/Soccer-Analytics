This is a personal project using open source football data from platforms such as Wyscout and StatsBomb to analyse various metrics of football performance analysis. 
KPIs include, but not limted to: expected goals, expected threats, win probability, long-term trends

**Aim** - The main focus of this project is to perform statistical analysis techniques to uncover patterns in a player,team or even a league 
      and be able to visualize in resourceful ways.

Primarily using libraries like statsmodels and matplotlib for analysis and visualisation.
